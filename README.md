# Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript.

## Overview

This project is a browser-based implementation of the classic Snake Game. Players control the snake using keyboard arrow keys, collect food to increase their score, and try to achieve the highest score possible without colliding with the walls.

## Features

* Real-time score tracking
* High score storage using Local Storage
* Food generation at random positions
* Keyboard-based controls
* Wall collision detection
* Game over screen
* Restart game functionality
* Built-in timer

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Play

1. Click the **Start Game** button.
2. Control the snake using the arrow keys.
3. Eat food to increase your score.
4. Avoid colliding with the walls.
5. Restart the game after a game over.
